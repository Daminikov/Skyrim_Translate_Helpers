# Скрипт 2: generate_prompts_to_AI_TXT.py
Генерирует готовые промпты для нейросетей (Grok, ChatGPT, Claude) на основе .txt файлов. Автоматически извлекает релевантный глоссарий из базы для каждого файла.

📋 Приоритеты словарей
При загрузке базы скрипт учитывает приоритеты:
Приоритет 1 — все обычные JSON-файлы
Приоритет 2 — Custom_Pet9948.json
Приоритет 3 — BDD_SkyrimSE_EN-RU.json
Приоритет 4 — User_Base.json (перезаписывает всё!)

🚫 Файл Exceptions.json
Слова из этого файла не попадают в глоссарий промпта.

📄 Шаблоны промптов
Шаблоны лежат в prompts/templates/. Первая строка — описание (игнорируется в промпте)

📋 Результат
В папке prompts/ появится файл {имя_файла}_PROMPT.txt:

💡 Использование
Откройте сгенерированный промпт
Скопируйте блок SYSTEM PROMPT в нейросеть (Grok, ChatGPT, Claude)
Скопируйте блок USER CONTENT вторым сообщением
Получите идеальный перевод с сохранением формата КЛЮЧ | Текст

# Скрипт 3: generate_prompts_to_AI_XML.py
Обрабатывает файлы переводов.xml из ESP-ESM Translator. Извлекает тексты из тегов <ORIGINAL>, генерирует промпт для перевода, а затем позволяет вставить переводы обратно в XML через теги <TRADUIT>.

📄 Шаблоны
Этот скрипт использует только шаблоны, начинающиеся на XML (например, XML_skyrim.txt). Остальные игнорируются.

🔄 Процесс работы
Шаг 1: Извлечение оригиналов
Скрипт читает XML файл и находит все теги <ORIGINAL>

<ORIGINAL>Thanks for downloading Completionist</ORIGINAL>
<TRADUIT />

Шаг 2: Генерация промпта
Для каждого найденного оригинала формируется строка с номером строки:
String_number_56 | Thanks for downloading Completionist
String_number_57 | Another text here

Затем извлекается глоссарий и формируется промпт (как в скрипте 2).
Шаг 3: Создание файла для перевода: В папке input/ создается пустой файл {имя}_translated.txt вставляем туда переведенный текст
Шаг 4: Вставка переводов обратно: После того как вы заполнили файл перевода, скрипт спрашивает:

Загрузить перевод в XML? (да/нет):

Если ответ "да", скрипт:
Читает файл из input/
Извлекает номер строки и перевод
Находит соответствующий тег <TRADUIT> в XML
Вставляет перевод: <TRADUIT>Ваш перевод</TRADUIT>

# Полезные советы
💡 User_Base.json — ваш главный инструмент
Если нейросеть переводит что-то неправильно, добавьте правильный перевод в User_Base.json. Этот файл имеет высший приоритет и перезапишет любые совпадения из других баз.
💡 Exceptions.json — чистый глоссарий
Если какие-то слова засоряют промпт и не нужны (например, "Note", "Valid Items"), добавьте их в Exceptions.json. Они не попадут в глоссарий, но нейросеть всё равно переведет их правильно в контексте.
💡 Шаблоны промптов
Создавайте разные шаблоны для разных задач:
skyrim.txt — стандартный перевод Skyrim
XML_skyrim.txt — для XML-файлов (игнорируется в txt-скрипте)
custom.txt — ваш экспериментальный шаблон
